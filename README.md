# Event Management System

This is a full-stack Event Management System built using the MERN-style architecture (React + Node.js + Express + SQLite).

The application allows users to register, login, browse events, register for events, and manage their registrations.

---

##  Features

- User Registration & Login (JWT Authentication)
- Browse all events
- Search events by name
- Filter events (location, category, date)
- Pagination support
- View event details
- Register for events
- Cancel event registration
- Dashboard to see registered events
- Seat management (available seats decrease/increase automatically)

---

##  Tech Stack

### Frontend
- React
- React Router
- Axios
- Bootstrap 5

### Backend
- Node.js
- Express.js
- SQLite
- JWT Authentication
